# IOS-Swift-MapKitAddAnnotation  
``` Level : Intermediate ```

This tutorial learn you how Add Annotation Pin on Map with LongPressGesture ( Latitude, Longitude). This tutorial is made with Xcode 10 and built for iOS 12.

## About

Sample app to Add Annotation on Map with LongPressGesture ( Latitude, Longitude)

## Requirements

- iOS 12.0+
- Xcode 10.0+
- Swift 4.0+

## Usage

You won't need to Download any extra libaray just ```import UIKit``` inside the project
